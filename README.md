### Introduction

LanguageOne is an Unreal Engine 5 plugin that enables one-click switching between Chinese and English editor languages.

### Features

- âœ… **One-Click Switching**: Quickly toggle between Chinese and English
- âŒ¨ï¸ **Customizable Shortcuts**: Default hotkey is `Alt + Q`
- ğŸš€ **Easy to Use**: Works out of the box
- ğŸ”Œ **Compatible**: Supports UE5.5 and above

### Installation

1. Download the plugin from [Fab](https://fab.com/s/dc840febb323) or [GitHub Releases](https://github.com/AniBullet/LanguageOne/releases)
2. Copy the `LanguageOne` folder to your project's `Plugins` directory
3. Launch your project and wait for compilation
4. Enable the plugin in `Edit > Plugins`
5. Restart the editor
